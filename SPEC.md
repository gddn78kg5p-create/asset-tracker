# 个人资产追踪 Web App - 规格说明

## 核心功能

### 1. 资产录入
- 物品名称
- 购买价格
- 购买日期
- 物品分类（数码/日用品/奢侈品/收藏品/其他）
- 备注

### 2. 日均成本计算
- 计算公式：(购买价格 - 当前估算价值) / 持有天数
- 或者：(购买价格 / 预计使用天数) * 已持有天数
- 显示：已使用天数、日均成本、预计剩余价值

### 3. 闲置流转记录
- 物品状态：使用中 / 闲置 / 已卖出
- 闲置物品标记
- 二手卖出价格记录

### 4. 二手卖出盈亏
- 记录卖出价格
- 自动计算：卖出价格 - 购买价格 = 盈亏
- 卖出日期

### 5. 资产总览看板
- 总资产金额
- 服役中物品数量
- 闲置物品数量
- 已卖出物品数量
- 总盈亏金额

## 技术方案

- **后端**：Python Flask
- **前端**：HTML5 + CSS3 + JavaScript
- **数据存储**：本地 JSON 文件
- **部署**：本地运行，通过局域网访问手机

## 页面布局

### Dashboard（首页）
- 顶部：总资产、闲置数、已卖出数
- 中间：最近添加的物品
- 底部：快捷操作按钮

### 物品列表
- 按分类显示
- 支持筛选（全部/使用中/闲置/已卖出）

### 添加/编辑物品
- 表单录入

### 物品详情
- 显示完整信息
- 日均成本计算
- 操作按钮（编辑/标记闲置/卖出/删除）
