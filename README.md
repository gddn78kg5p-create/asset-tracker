# 个人资产追踪 Web App

一个仿"有数"App的个人资产管理系统，完全免费，数据本地存储。

## 功能

- ✅ 资产录入（名称/价格/日期/分类）
- ✅ 日均成本计算
- ✅ 闲置流转记录
- ✅ 二手卖出盈亏计算
- ✅ 资产总览看板
- ✅ 数据导出/导入（JSON格式）

## 使用方法

### 在电脑上使用
1. 双击打开 `index.html`
2. 或者拖入浏览器打开

### 在手机上使用（推荐添加到主屏幕）
1. 用手机浏览器打开 `index.html`
2. Safari: 点击分享按钮 → "添加到主屏幕"
3. Chrome: 点击菜单 → "安装应用" 或 "添加到主屏幕"
4. 之后可以像App一样从主屏幕打开

## 数据存储

- 数据保存在浏览器**本地存储(localStorage)**
- 不会上传到任何服务器，完全隐私
- **重要**: 清除浏览器缓存会导致数据丢失！
- 定期点击右上角"📤 导出"备份数据

## 数据备份

- 点击右上角 **📤 导出** 按钮，下载 JSON 文件
- 换浏览器或设备后，点击 **📥 导入** 按钮选择之前导出的文件
